\# 架构设计规约：v2.0 全域机动期



\*\*1. 核心目标\*\*

放开初始条件，逼迫网络在长距离机动中遭遇 SGCMG 奇异面。通过训练，使其学会主动调动飞轮加减速（真正的 VSCMG 模式），利用动量交换强行越过奇异构型。



\*\*2. 环境初始化 (env.reset)\*\*

\* \*\*姿态误差\*\*：初始四元数/欧拉角在全包络域内随机生成（±180°）。

\* \*\*角速度\*\*：赋予随机的初始翻滚角速度，增加 Detumbling 难度。



\*\*3. 动力学与动作流转 (env.step)\*\*

\* \*\*边界感知探索噪声\*\*：当检测到飞轮逼近物理极限（5000 rpm）时，强行截断加剧饱和方向的随机探索噪声，防止网络早期“自杀”。

\* \*\*动作平滑惩罚\*\*：在 Reward 中激活一阶差分惩罚（基于 t 与 t-1 时刻动作的欧氏距离），严厉打击框架高频打舵。



\*\*4. 验收标准\*\*

\* 卫星能平顺完成 180° 大机动。

\* 机动轨迹中必须观测到：遭遇奇异时飞轮转速短促拉升（借力），到位后自动回落至基准附近。
